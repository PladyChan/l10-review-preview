# Panasonic Lumix L10 review preview

Static reading preview for the Panasonic Lumix L10 review outline.

- `index.html`: mobile-first reading preview.
- `editor.html`: mobile-first Markdown editor for `source/L10_测评对比草稿.md`.
- `build_l10_html_preview.py` and `.github/workflows/rebuild-preview.yml`: rebuild `index.html` automatically after the Markdown source changes.
